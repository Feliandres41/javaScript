Ejercicio #1

Realice un programa que sirva para determinar cuál es el cuadrado, de un determinado número.

Ejercicio #2

Realice un programa que sirva para obtener el promedio simple de un estudiante a partir del ingreso de las tres notas obtenidas durante el primer periodo académico.

Ejercicio #3

Realice un programa que solicite la edad de dos hermanos y muestre un mensaje indicando la edad del mayor y cuantos años de diferencia tiene con el menor.

Ejercicio #4

Realice un programa que permita calcular el área de un triángulo aplicando la siguiente formula: A = (B x H) /2.

Ejercicio #5

Construya un pseudocódigo que, dado el nombre de una persona, su peso y su altura, expresados estos dos últimos en libras y pies, respectivamente; escriba el nombre de la persona, su peso expresado en kilogramos y su altura expresada en metros.
